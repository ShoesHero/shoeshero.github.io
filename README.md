ShoesHero — Portfolio
=====================

This is a custom React + Vite portfolio for showcasing your work with rich animations and a built-in music player.

## Run locally

- **Install dependencies**: `npm install`
- **Start dev server**: `npm run dev` then open the printed URL (by default `http://localhost:5173`)

## Customizing content

- **Experience timeline**: Edit the `EXPERIENCE` array in `src/components/ExperienceSection.tsx`.
- **Playlist**: Update the `PLAYLIST` array in `src/components/MusicPlayer.tsx` and place your `.mp3` files in `public/audio/`.
